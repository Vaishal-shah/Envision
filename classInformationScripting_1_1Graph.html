using InformationScripting::Graph::NodeCondition = std::function<bool(const InformationNode*)>
using InformationScripting::Graph::NodeHash = std::function<QPair<std::size_t, bool>(const InformationNode*)>

Each information source can register 1 NodeHash function.

The function takes a node as input, and should return a pair of the hash value and a bool if the function could succesfully hash the value.

The second parameter is useful as, e.g. a node from the tag information source can not be hashed by the ast information source.

Creates an Edge from from to to with the name name.

Returns the Edge with orientation orientation between from and to.

If there is already an edge with the same name and orientation between from and to, the existing edge is returned with the count property increased.

Note: The graph owns the returned edge.
